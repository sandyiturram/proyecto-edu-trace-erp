Manual de Capacitación EDU-TRACE ERP

Bienvenido al manual interactivo diseñado para profesores y facilitadores. Este documento le guiará a través del funcionamiento lógico del sistema, diseñado específicamente para la enseñanza de procesos de negocio y contabilidad.

Vista principal del Dashboard con indicadores clave.

¿Qué es EDU-TRACE?

Es un ERP educativo que prioriza la trazabilidad pedagógica. A diferencia de un software contable tradicional, EDU-TRACE explica el origen de cada movimiento, permitiendo al estudiante entender no solo los números, sino el flujo de información que los genera.

Nota para el Docente: El sistema permite dos modos de trabajo: Directo (asientos inmediatos) y Centralizado (vía libros auxiliares), configurable en el módulo de Administración.

Primeros Pasos en el Sistema

Antes de comenzar cualquier operación comercial, el sistema requiere una estructura básica.

1

Gestión de Empresas

Puede crear múltiples escenarios de práctica para sus alumnos. Cada empresa es independiente y guarda su propia base de datos local.

2

Importación de Datos Maestro

Para simplificar el inicio de clases, EDU-TRACE incluye un botón de "Datos de Ejemplo" que precarga un plan de cuentas estándar, clientes, proveedores y productos.

El Ciclo de Compras (MM)

El proceso comienza con la necesidad de adquirir mercaderías para el negocio.

1

Generación de OC

Se crea una Orden de Compra seleccionando un proveedor y los productos. El sistema valida el stock proyectado.

Gestión de órdenes con estados interactivos.
2

Recepción de Mercaderías

Al hacer clic en el ícono del Camión (🚛), el sistema simula la llegada del pedido a bodega, incrementando automáticamente el stock en el inventario.

El Ciclo de Ventas (SD)

La operación lógica que genera ingresos para la compañía.

1

Pedido de Venta

Seleccione el cliente y defina los artículos. El sistema verifica disponibilidad de stock real antes de confirmar.

Control de pedidos y despacho a clientes.
2

Despacho y Facturación

Una vez entregado, se habilita el ícono de Documento (📄) para generar la factura. Este paso es CRUCIAL, ya que dispara la integración contable.

Integración Contable Automática

La "magia" de EDU-TRACE: Todo proceso operativo tiene un reflejo contable inmediato y trazable.

1

Libro Diario

Cada vez que factura una venta o compra, el sistema genera el asiento contable correspondiente (Debe/Haber) con su glosa explicativa.

Traceabilidad directa: de la factura al asiento contable.
Tip Pedagógico: Pida a sus alumnos que revisen el asiento generado después de cada venta para validar que se reconoció el Costo de Venta y la baja de Inventario.

Reportes y Toma de Decisiones

El sistema consolida toda la información en informes financieros estándar.

  • Balance General: Situación financiera de activos, pasivos y patrimonio.
  • Estado de Resultados: Análisis de rentabilidad (ingresos vs gastos).
  • Ratios Financieros: Liquidez, solvencia y rentabilidad calculados automáticamente.
